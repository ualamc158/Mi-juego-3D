3. TROUBLESHOOTING: "MY MODELS ARE PINK!"
-----------------------------------------------------------
If the models appear bright pink in your scene, it means 
your project is not yet configured for the Universal 
Render Pipeline (URP) or the material needs a quick refresh.

Follow these steps to fix it:

A. Ensure URP is Installed:
   Go to Window > Package Manager and ensure "Universal RP" 
   is installed.

B. Update the Material:
   1. Select the "M_Goblin" material in your Project window.
   2. In the Inspector, click the "Shader" dropdown.
   3. Change it to: Universal Render Pipeline > Lit (or Simple Lit).

C. Global Project Upgrade (Recommended):
   If your whole scene is pink, go to:
   Edit > Rendering > Materials > Convert Selected Built-in 
   Materials to URP.
